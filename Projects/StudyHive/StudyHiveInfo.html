What is StudyHive ?

  • Notion-inspired app for smarter studying.
  • Hosted and deployed on Firebase
  • Languages ReactJS
    TailwindCSS

This Notion-inspired web application, currently under development, allows users to create an account, log in, and access a personalized dashboard. Within this dashboard, users can visualize all their existing classes and easily create new ones. Each class serves as a space where users can upload and organize a variety of documents. The application is fully hosted and deployed using Firebase, offering a seamless experience for students to manage their academic materials efficiently.

A working link to the application will be provided once the project is completed.

What is the Insulin Calculator?

  • Helps to automate insulin dosage calculations for patients.
  • Helpful for doctors diagnosing new cases of diabetes.
  • Script Python
  • Demo JS/HTML/CSS

This project was created to automate the process of calculating insulin dosages for newly diagnosed diabetes patients. The tool enables residents and doctors to perform dosage calculations independently and then verify their results using the application. It was specifically developed to assist several pediatric resident doctors at CHUS Fleurimont in Sherbrooke, Quebec.

The main script was developed in Python, with a demo version created using JavaScript, HTML, and CSS, allowing easy access through my website.
